React에서는 HTML 대신 JSX를 사용한다!


## JSX
---

```javascript
function App(){
	return ( 
		<div>html소스</div>
	)	 
}
```
App.js 의 App함수에 HTML소스를 return하면 됨!
<br/>

#### class
    JSX_에서는 class가 아닌 className을 사용함
```bash
npm start
```
