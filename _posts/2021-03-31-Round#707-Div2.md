# A. Alexey and Train
---
[문제보기](https://codeforces.com/contest/1501/problem/A)

단순 Bruteforce 문제
<details>
<summary>코드 보기/숨기기</summary>
<div markdown="1">

```c++
#include <bits/stdc++.h> 
#include <fstream>
 
using namespace std;
 
int t, n;
int a[150], b[150], tmm[150];
 
int main(void) {
    //freopen("input.txt", "r", stdin);
    ios_base::sync_with_stdio(false);
    cin.tie(NULL); cout.tie(0);
    
    cin >> t;
   
    while(t--) {
        b[0] = 0;
        int now = 0;
        cin >> n;
        for(int i = 1; i <= n; i++)
            cin >> a[i] >> b[i];
        for(int i = 1; i <= n; i++)
            cin >> tmm[i];
 
        for(int i = 1; i <= n; i++) {
            now += a[i] - b[i - 1] + tmm[i] + (int)ceil((float)(b[i] - a[i]) / 2);
            if(i == n) now -= (int)ceil((float)(b[i] - a[i]) / 2);
            if(i != n && now < b[i]) now = b[i];
        }
        cout << now << "\n";
    }
    
    return 0;
}
 
```

# B. Napoleon Cake
---
[문제보기](https://codeforces.com/contest/1501/problem/B)

cake의 윗 부분부터 적셔간다.

 $a_n$번째에서 적실 수 있는 layer부터 적셔온 
<details>
<summary>코드 보기/숨기기</summary>
<div markdown="1">

```c++
#include <bits/stdc++.h> 
#include <fstream>
 
using namespace std;
 
int t, n;
int a[200010], ans[200010];
 
int main(void) {
    freopen("input.txt", "r", stdin);
    ios_base::sync_with_stdio(false);
    cin.tie(NULL); cout.tie(0);
    
    cin >> t;
   
    while(t--) {
        cin >> n;
        for(int i = 1; i <= n; i++)
	        cin >> a[i];
        
        int cnt = n;
        for(int i = n; i > 0; i--) {
            cnt = min(cnt, i - min(a[i], i));
            ans[i] = i > cnt ? 1 : 0;
        }
        for(int i= 1; i <= n; i++)
            cout << ans[i] << " ";
        cout << "\n";
    }
    return 0;
}
```
