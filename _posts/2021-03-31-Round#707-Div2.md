## A. Alexey and Train
---
[문제보기](https://codeforces.com/contest/1501/problem/A){:target="_blank"}

- 단순 Bruteforce 문제
<details>
<summary>코드 보기/숨기기</summary>
<div markdown="1">

```c++
#include <bits/stdc++.h> 
#include <fstream>
 
using namespace std;
 
int t, n;
int a[150], b[150], tmm[150];
 
int main(void) {
    //freopen("input.txt", "r", stdin);
    ios_base::sync_with_stdio(false);
    cin.tie(NULL); cout.tie(0);
    
    cin >> t;
   
    while(t--) {
        b[0] = 0;
        int now = 0;
        cin >> n;
        for(int i = 1; i <= n; i++)
            cin >> a[i] >> b[i];
        for(int i = 1; i <= n; i++)
            cin >> tmm[i];
 
        for(int i = 1; i <= n; i++) {
            now += a[i] - b[i - 1] + tmm[i] + (int)ceil((float)(b[i] - a[i]) / 2);
            if(i == n) now -= (int)ceil((float)(b[i] - a[i]) / 2);
            if(i != n && now < b[i]) now = b[i];
        }
        cout << now << "\n";
    }
    
    return 0;
}
 
```

## B. Napoleon Cake
---
[문제보기](https://codeforces.com/contest/1501/problem/B){:target="_blank"}


- cake의 윗 부분부터 적셔간다
- 각 층마다 가능한 아랫부분까지 적신다
-  현재 층이 적셔져있는지 확인한다
- *O(n)*
<details>
<summary>코드 보기/숨기기</summary>
<div markdown="1">

```c++
#include <bits/stdc++.h> 
#include <fstream>
 
using namespace std;
 
int t, n;
int a[200010], ans[200010];
 
int main(void) {
    freopen("input.txt", "r", stdin);
    ios_base::sync_with_stdio(false);
    cin.tie(NULL); cout.tie(0);
    
    cin >> t;
   
    while(t--) {
        cin >> n;
        for(int i = 1; i <= n; i++)
	        cin >> a[i];
        
        int cnt = n;
        for(int i = n; i > 0; i--) {
            cnt = min(cnt, i - min(a[i], i));
            ans[i] = i > cnt ? 1 : 0;
        }
        for(int i= 1; i <= n; i++)
            cout << ans[i] << " ";
        cout << "\n";
    }
    return 0;
}
```

##  C. Going Home
---
[문제보기](https://codeforces.com/contest/1501/problem/C){:target="_blank"}

- $a_i$과 $a_j$ $(i>j)$의 합을 인덱스로 하는 배열 원소가 비어있다면 $i$와 $j$를 저장한다.
- 비어있지 않다면 $a_x+a_y=a_z+a_w$를 만족하므로 출력한다.
- 여기까지 보면 $O(n^2)$인데? 2초안에 못푸는데요??
<br>
- $a_i\leq 2.5e6$
- 최악의 경우 $a_x+a_y\neq a_z+a_w$합 배열이
<details>
<summary>코드 보기/숨기기</summary>
<div markdown="1">

```c++
#include <bits/stdc++.h> 
#include <fstream>
 
using namespace std;
 
int n;
int a[200010], sumX[5000010], sumY[5000010];

int main(void) {
    freopen("input.txt", "r", stdin);
    ios_base::sync_with_stdio(false);
    cin.tie(NULL); cout.tie(0);
    
    cin >> n;
    for(int i = 1; i <= n; i++) cin >> a[i];

    // answer always exist when all array has filled (pigeonhole principle)
    // so time complexity is not n^2,
    // O(min(n^2, 5000000))
    for(int i = 1; i <= n; i++) {
        for(int j = i + 1; j <= n; j++) {
            int &x = sumX[a[i] + a[j]], &y = sumY[a[i] + a[j]];
            if(x == 0)
                x = i, y = j;
            else 
                if(x != i && y != i && x != j && y != j) {
                    cout << "YES\n"<< x << " " << y << " " << i << " " << j;
                    return 0;
                }
        }
    }
    cout << "NO";
    return 0;
}
```
</div>
</details>
