React에서는 Ajax를 쉽게 사용할 수 있도록 axios 라이브러리를 제공한다.

## axios
---
#### 설치
`npm install axios`

#### 사용
```jsx
import { axios } from  'axios';

axios.get(...)
.then((result)=>{...}) //성공
.catch(()=>{...}); //실패

axios.post(...);
```
