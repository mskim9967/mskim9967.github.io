중첩된 Components에서 props로 state를 넘기는 과정은 상당히 귀찮다.
이럴때 사용할 수 있는 라이브러리 redux를 사용해보자.

## redux
---
#### 설치
`npm install redux react-redux`

#### 사용
```jsx
// index.js
import { Provider } from  'react-redux';
import { createStore } from 'redux';

let defaultState = ["aa", "bb"];

function reducer(state = defaultState,action) {
	if(action.type === 'change') {
		...
		action.payload..
		...
		return ...
	}
	return state
}

let store = createStore(reducer);

ReactDOM.render(
	<React.StrictMode> 
		<BrowserRouter>
			<Provider store={store}>
				<App/> 
			</Provider> 
		</BrowserRouter>
	</React.StrictMode>
);
```
```jsx
// Comp.js
import { connect } from 'react-redux'

function Comp(props) {
	return (
		{ props.state[0] }
	)
}

function stateToProps(state) {
	return {
		state : state
	}
}

export default connect(stateToProps)(Comp)
```
state 변경을 위해서는 `dispatch()` 함수를 사용한다.
```jsprops.dispatch({type: 'change', payload: ...})`
